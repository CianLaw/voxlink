为什么选择 VoxLink

为追求极致效率的开发者、创作者和职场人士打造

高精度语音识别

采用 Silero VAD 人声检测引擎，精准识别人声起止。结合大模型智能纠错，自动修正同音字、补全标点符号。

灵动岛悬浮设计

精致毛玻璃悬浮条，不遮挡工作区域。支持全屏、多屏、多空间场景，始终置顶但不抢夺焦点。

无感文本注入

智能备份剪贴板，注入后自动恢复。支持 macOS 辅助功能和 Windows UIAutomation，精确定位光标位置。

技术架构

基于 Rust + Tauri v2 构建，极致性能与安全性

音频引擎

  • - CPAL 无锁环形队列音频采集
  • - Rubato SincFixedIn 高精度重采样至 16kHz
  • - Silero VAD 实时人声激活检测
  • - Windows 11 24H2 WASAPI 驱动层兼容

文本注入

  • - macOS AXUIElement + CGEvent 键盘模拟
  • - Windows IUIAutomation8 + SendInput 扫描码
  • - 剪贴板快照备份与静默恢复
  • - 100ms 阻尼时间确保粘贴完成

常见问题

macOS 上提示"应用已损坏"怎么办？

VoxLink 已通过 Apple 公证，不会触发 Gatekeeper 警告。如果仍然提示，请在终端执行：xattr -cr /Applications/VoxLink.app

需要授予哪些系统权限？

macOS 需要授予「辅助功能」和「麦克风」权限。首次启动时会自动引导您打开系统设置。Windows 需要以管理员身份运行安装程序。

语音数据会上传到云端吗？

不会。所有语音识别和处理均在本地完成。仅当您启用大模型纠错功能时，文本会发送到您配置的 LLM API 端点。

支持哪些语言的语音输入？

目前支持中文（普通话）和英文。更多语言支持正在开发中。
